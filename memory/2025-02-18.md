## 项目愿景

健康数据管理和共享平台，帮助每个人都能拥有对健康或疾病的数据和信息归纳和总结的能力，同时也能把自己健康或者疾病康复的案例分享给其他人。

## 核心需求

### 数据归纳和总结

#### 支持的数据类型
- 病历
- 出院报告
- 体检报告
- 中医报告
- 线上检验报告
- 穿戴设备数据
- 口述记录
- 视频记录
- 健康报告

#### 支持的数据格式
- 图片
- 文本
- 声音
- 视频
- API 接口数据（穿戴设备）
- 网络链接

#### 数据输入方式
- 拍照上传
- 文件上传
- 录音录像
- 线上链接导入

#### 数据输出形式
- 可视化图表
- 文字总结
- 趋势分析

### 案例分享

#### 分享内容
- 完整病历
- 特定疾病康复过程
- 生活方式改变

#### 分享形式
- 类似抖音的视频分享形式

#### 隐私和匿名化
- 用户自主决定是否匿名
- 用户自主决定是否分享

#### 收费模式
- 每次被查阅：分享者获得5元，查阅者支付6元，平台收取1元
- 前十次查阅免费
- 从第11次开始收费

### 信息交流和沟通

#### 交流方式
- 虚拟号码联络
- 文字交流
- 视频交流

#### 收费标准
- 3分钟内：6元知识付费
- 超出时间：每分钟3元
- 自动记录交流时间
- 预付费方式
- 超时自动断线

### 工具形态

#### 应用平台
- Web 应用
- 移动应用（iOS + Android）

#### 存储和同步
- 需要考虑离线功能
- 需要考虑数据本地化存储
