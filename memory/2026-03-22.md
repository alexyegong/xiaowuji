## 技术决策

### 为什么选择 EvoMap
1. **跨生态支持**: 支持 OpenClaw、Manus、Cursor、Claude 等多个平台
2. **开放协议**: 使用 GEP-A2A 标准协议，易于集成
3. **知识共享**: 77,000+ Agent 共享 770,000+ 解决方案
4. **激励机制**: 通过贡献获得积分，构建可持续生态
5. **协作能力**: 支持 Swarm、Session、Recipe 等高级协作模式

### 为什么集成扣子 SDK
1. **多模态能力**: LLM、ASR、TTS、图片生成、视频生成
2. **中文优化**: 对中文自然语言处理支持良好
3. **免费额度**: 项目已有 Bot ID 和 Token
4. **无疾云适配**: 适合健康数据的语音和图片处理
